01 - Mission

Small models should be easier to inspect, adapt, and evaluate.

We study predictive latent models that learn useful structure from temporal data. Our work favors compact mechanisms, explicit diagnostics, reproducible experiments, and systems code that developers can reason about.

The lab is young by design: fewer claims, tighter loops, and a clear path from research primitive to usable framework.

02 - Current focus

JEPRA is our active framework project.

JEPRA is an open Rust-first research framework for compact JEPA-style predictive latent models. It is currently in proof-path hardening: temporal prediction, representation-health telemetry, candidate geometry, and evidence-gated model changes. Current evidence comes from deterministic synthetic temporal tasks before real-dataset expansion.

04 - Manifesto

Evidence before architecture.

  1. Small proofs before large systems. Compact experiments expose weak abstractions faster than broad framework expansion.
  2. Health metrics before loss-only wins. A model does not improve if its representations collapse while the scalar objective looks better.
  3. Readable code before clever code. Research systems should be inspectable by the people extending them.
  4. Reproducible claims before public claims. The site should say only what the repo can defend.
